# EchoWord

轻量级跨平台划词翻译工具，核心差异化功能：音标中文谐音标注。

## 技术栈

- **框架：** Tauri v2 (Rust + React)
- **前端：** React + TypeScript + Vite
- **样式：** Tailwind CSS
- **组件库：** shadcn/ui
- **状态管理：** zustand
- **数据库：** SQLite (rusqlite)
- **离线词典：** ECDICT（核心版 5 万词内置）
- **目标平台：** macOS、Windows (P0)，Linux (P2)

## 关键决策记录

1. **翻译源：** 可配置，离线词典（单词首选）+ 腾讯/百度/有道 API（句子翻译）
2. **音标谐音：** 混合标注方案——能对应的用中文字，无法对应的保留音标+发音提示
3. **数据同步：** 数据目录可配置到坚果云同步文件夹，利用坚果云客户端文件同步
4. **弹窗交互：** 鼠标位置气泡弹窗，点击外部/Esc 关闭，不超时消失
5. **触发方式：** 全局快捷键 + 鼠标选中自动弹窗，两者都支持
6. **语言：** 当前仅英→中，架构预留多语言扩展
