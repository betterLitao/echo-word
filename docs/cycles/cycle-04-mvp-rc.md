# 开发周期 04：P0 收口与发布候选

## 周期目标

补齐所有 `P0` 要求，完成权限引导、鼠标选中自动弹窗、设置闭环和稳定性加固，形成可对外测试的 MVP Release Candidate。

## 本周期范围

### 必做功能

- 实现鼠标选中自动弹窗翻译。
- 完成首次启动引导、权限申请和引导完成状态落库。
- 完成弹窗关闭规则、弹窗复用和跟随定位。
- 完成翻译源切换、SQLite 本地存储、Migration 回滚与异常处理收口。
- 完成系统托盘与主窗口行为收口，保证“关闭即隐藏到托盘”。
- 完成 `P0` 全量回归与发布候选版本检查清单。

### 目标模块

- 平台：`src-tauri/src/utils/platform.rs`
- 命令：`src-tauri/src/commands/settings.rs`、`src-tauri/src/commands/translate.rs`
- 前端引导：`src/components/onboarding/*`
- 前端弹窗：`src/components/popup/PopupWindow.tsx`
- 数据层：`src-tauri/src/db/migration.rs`
- 应用入口：`src-tauri/src/main.rs`

## 开发任务

### 系统能力

- 落地 macOS 辅助功能权限检测与跳转系统设置能力。
- 建立选中文本监听或兼容方案，满足“选中即自动弹窗”的产品要求。
- 完成弹窗定位逻辑，保证在鼠标附近显示且不频繁闪烁。

### 前端交互

- 完成 5 步引导流程：欢迎、权限、翻译源、快捷键、完成。
- 引导完成后写入 `onboarding_completed = true`，后续启动直接进入主设置页。
- 实现弹窗点击外部关闭、`Esc` 关闭、显示/隐藏复用。
- 完成设置页中 `P0` 必需项的可编辑闭环。

### 稳定性与发布

- 加固数据库 Migration：备份、事务、失败回滚、日志记录。
- 补齐翻译失败、权限未授权、剪贴板为空、网络不可达等异常态提示。
- 对离线单词翻译、在线句子翻译、快捷键、弹窗、收藏做一次完整 P0 回归。
- 形成 MVP 测试清单和发布前检查清单。

### 验证清单

- 首次启动会进入引导，完成后不会重复出现。
- 权限未开启时能正确提示并跳转系统设置。
- 用户在任意应用中选中文本后可以触发自动弹窗。
- 弹窗位置、关闭方式、复用策略符合 PRD 说明。
- PRD 中 `P0 - MVP 必须` 列表项可逐条验收。

## 交付物

- `P0` 功能完整的 MVP 候选版本。
- 首次启动引导与权限流程。
- 完整回归通过的发布前状态。

## 验收标准

- `docs/PRD.md` 中 `P0 - MVP 必须` 项全部具备实现与可验证路径。
- 关键性能目标基本达成：弹窗响应、离线查询、缓存命中可达到设计级别。
- 权限缺失、翻译失败、网络失败等异常路径均不会导致应用崩溃。

## 非本周期范围

- 发音朗读
- 历史记录与收藏列表页
- 深色主题
- 词典下载管理
- OpenAI、多引擎、HTTP API、自动更新

## 风险与依赖

- “鼠标选中自动弹窗”是整个 `P0` 中实现风险最高的系统集成点，应优先验证可行性。
- 权限引导如果设计不清晰，会直接影响首次安装成功率。
- 本周期必须控制新增需求，避免 `P0` 迟迟无法冻结。

## 建议 PR 拆分

1. 权限检测、系统设置跳转、选中自动触发方案
2. 引导页与首次启动状态流转
3. 弹窗定位、关闭、复用与异常提示
4. P0 回归修复与发布检查清单
