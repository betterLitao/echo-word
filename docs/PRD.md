# EchoWord - 产品需求文档 (PRD)

## 1. 产品概述

**产品名称：** EchoWord

**定位：** 轻量级跨平台划词翻译工具，核心差异化功能为音标中文谐音标注，帮助不熟悉国际音标的用户快速理解英语发音。

**技术栈：** Tauri (Rust + Web 前端)

**目标平台：** macOS、Windows

**当前语言支持：** 英 → 中（架构预留多语言扩展能力）

---

## 2. 核心功能

### 2.1 翻译触发方式

| 触发方式 | 说明 | 优先级 |
|---------|------|--------|
| 全局快捷键 | 选中文本后按快捷键触发翻译，读取系统剪贴板 | P0 |
| 鼠标选中自动弹窗 | 在任意应用中选中文本后自动弹出翻译气泡（需辅助功能权限） | P0 |
| 输入翻译 | 通过快捷键唤出输入框，手动输入文字查询 | P0 |

**适用场景：** 浏览器、IDE（VSCode 等）、微信、任意桌面应用。

### 2.2 翻译能力

#### 2.2.1 智能模式判断

- 单个单词（无空格） → **单词模式**：显示释义 + 音标 + 谐音
- 多个单词 → **句子模式**：显示整句翻译
- 弹窗内提供手动切换按钮，允许在两种模式间切换

#### 2.2.2 单词翻译

- **数据来源：** 离线词典（优先）或有道词典 API
- **展示内容：**
  - 英文单词
  - 中文释义
  - 国际音标（美式/英式）
  - 音标中文谐音标注（混合方案 B）

#### 2.2.3 句子翻译

- **数据来源：** 在线翻译 API（腾讯/百度/有道，用户可配置）
- **展示内容：**
  - 原文
  - 中文翻译

### 2.3 音标中文谐音标注（核心差异化功能）

采用 **混合标注方案**：

- 能对应的音标：用最近似的中文字标注
- 无法对应的音标：保留原始音标符号 + 发音提示文字

示例：

```
ephemeral /ɪˈfemərəl/
谐音：一 · 飞 · 摸 · 若 · 了

think /θɪŋk/
谐音：[θ咬舌送气] · 英 · 克
```

## 3. 翻译源管理

### 3.1 离线词典

| 版本 | 词条数 | 体积 | 说明 |
|------|--------|------|------|
| 核心版 | 5 万 | ~50MB | 内置，覆盖日常常用词 |
| 精简版 | 20 万 | ~200MB | 可选下载 |
| 完整版 | 77 万 | ~800MB | 可选下载 |

- 基于 ECDICT 开源词典
- 默认内置核心版，精简版和完整版通过应用内词典管理页面一键下载
- 离线词典查询失败时，自动降级到在线 API

### 3.2 在线翻译 API

| API | 音标支持 | 免费额度 | 用途 |
|-----|---------|---------|------|
| 腾讯翻译君 | ❌ | 500 万字符/月 | 句子翻译（默认） |
| 百度翻译 | ❌ | 标准版免费（QPS=1） | 句子翻译（备选） |
| 有道词典 | ✅ | 体验金 50 元 | 单词翻译（备选） |

### 3.3 配置能力

- 用户可在设置页面切换翻译源
- 支持离线/在线模式切换
- 用户自行填写 API Key

---

## 4. 弹窗交互

### 4.1 气泡弹窗

- **位置：** 跟随鼠标，出现在选中文字附近
- **关闭方式：**
  - 点击弹窗外部区域
  - 按 Esc 键
- **不做超时自动消失**（用户需要时间查看谐音、点收藏、点复制）

### 4.2 弹窗内容（单词模式）

```
┌─────────────────────────────────┐
│ ephemeral            🔊 ⭐ 📋  │
│ /ɪˈfemərəl/                    │
│ 谐音：一 · 飞 · 摸 · 若 · 了    │
│─────────────────────────────────│
│ adj. 短暂的；转瞬即逝的          │
│                                 │
│ [单词模式 | 句子模式]            │
└─────────────────────────────────┘
```

### 4.3 弹窗内容（句子模式）

```
┌─────────────────────────────────┐
│ I love coding            🔊 📋 │
│─────────────────────────────────│
│ 我喜欢编程                      │
│                                 │
│ [单词模式 | 句子模式]            │
└─────────────────────────────────┘
```

## 5. 收藏功能

### 5.1 收藏操作

- 弹窗内点击 ⭐ 一键收藏
- 收藏数据结构：

```json
{
  "word": "ephemeral",
  "phonetic": "/ɪˈfemərəl/",
  "chinese_phonetic": "一·飞·摸·若·了",
  "translation": "adj. 短暂的；转瞬即逝的",
  "starred": true,
  "created_at": "2026-03-08T10:30:00Z",
  "review_count": 0
}
```

### 5.2 收藏列表

- 查看所有收藏的单词
- 支持搜索（按单词、释义搜索）
- 优先级：P1

### 5.3 导出

- 支持导出为 CSV、JSON、Anki 卡片格式
- 优先级：P2

---

## 6. 历史记录

- 自动记录每次翻译内容（被动记录，无需用户操作）
- 方便回溯查找
- 优先级：P1

---

## 7. 发音朗读

- 弹窗内点击 🔊 播放发音
- 支持单词和句子朗读
- 实现方式：系统 TTS 或在线发音 API
- 优先级：P1

---

## 8. 数据存储与同步

### 8.1 本地存储

- 使用 SQLite 存储所有数据（收藏、历史、设置）
- 优先级：P0

### 8.2 数据同步

- 应用数据目录支持用户自定义路径
- 将数据目录设置到坚果云同步文件夹，即可实现多设备同步
- 不需要应用内实现同步逻辑，利用坚果云客户端的文件同步能力
- 优先级：P1

---

## 9. 系统集成

### 9.1 系统托盘

- 关闭窗口 → 最小化到系统托盘（非退出）
- 托盘图标右键菜单：显示主窗口、设置、退出
- 优先级：P0

### 9.2 开机自启

- 可选开机自动启动
- 优先级：P2

---

## 10. 设置

| 设置项 | 说明 | 优先级 |
|--------|------|--------|
| 快捷键自定义 | 自定义全局翻译快捷键 | P1 |
| 翻译源切换 | 离线/在线切换，选择 API 提供商 | P0 |
| API Key 配置 | 用户填写自己的翻译 API Key | P0 |
| 数据目录 | 自定义数据存储路径 | P1 |
| 主题 | 深色/浅色模式，跟随系统 | P1 |

## 11. 截图翻译（P2）

- 通过快捷键触发截屏
- OCR 识别截图中的文字
- 翻译识别结果
- 优先级：P2，MVP 阶段暂不实现

---

## 12. 优先级总览

### P0 - MVP 必须

- 全局快捷键触发翻译
- 鼠标选中自动弹窗翻译
- 输入翻译
- 单词翻译（离线词典）
- 句子翻译（在线 API）
- 音标显示 + 混合谐音标注
- 智能单词/句子模式判断 + 手动切换
- 气泡弹窗（跟随鼠标）
- 一键收藏
- 翻译源配置切换
- SQLite 本地存储
- 系统托盘常驻

### P1 - 体验增强

- 发音朗读
- 收藏列表查看/搜索
- 翻译历史记录
- 翻译结果一键复制
- 快捷键自定义
- 深色/浅色主题
- 数据目录可配置（坚果云同步）
- 词典管理（下载精简/完整版）
- 百度翻译 API 支持
- 有道词典 API 支持

### P2 - 后续迭代

- 截图 OCR 翻译
- 收藏导出（CSV/JSON/Anki）
- 开机自启

---

## 13. 非功能性需求

| 类别 | 要求 |
|------|------|
| 性能 | 离线词典查询 < 50ms，弹窗响应 < 200ms |
| 体积 | 安装包 < 30MB（不含词典扩展包） |
| 内存 | 常驻内存 < 80MB |
| 架构 | 预留多语言扩展能力（当前仅英→中） |
| 安全 | API Key 本地加密存储，不上传 |
